# COMP3133 - Lab Test 2 - 101186647

## Description

An Angular application that display Harry Potter characters and details fetched from https://hp-api.onrender.com/

## Features implemented

- List of all Harry Potter characters using the provided REST API endpoint
- Allows users to search/filter characters by house
- Display detailed information about a selected character from the character list
